• Dashboard

    The dashboard is what you are looking at now and it is the central place from which we can control the IDE and our projects. It can be toggled on/off easily with the alt+~ key. When working on code, it will not get in the way until you actually need it.

  • Starting a project

    The simplest course of action is to go to the New Project item on the left side of the screen. Follow its instructions and you can begin writing code. Use Open Project to continue working on an existing one.
